## ğŸ”§ Troubleshooting

### Game Won't Launch
- **Solution**: Run as Administrator
- Check Windows version compatibility
- Update graphics drivers
- Disable antivirus temporarily during install

### Installation Fails
- **Solution**: Extract ZIP to a folder without special characters
- Ensure you have admin rights
- Check available disk space
- Download fresh copy if file is corrupted

### Performance Issues
- **Solution**: Close other applications
- Lower graphics settings in-game
- Update graphics drivers
- Check system requirements

### Save Game Issues
- **Solution**: Don't delete game files manually
- Ensure proper shutdown (don't force close)
- Backup saves from `%APPDATA%/RetroBowl/`

### Black Screen on Launch
- **Solution**: Update DirectX
- Update graphics drivers
- Run in compatibility mode (Windows 8)
- Disable fullscreen optimization
